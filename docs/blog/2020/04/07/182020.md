# VS Code Meetup #4 #vscodejp で「Remote Development(ssh)小技集」というテーマでLTを行いました。

2020/4/7にオンラインで開催された「VS Code Meetup #4」で「Remote Development(ssh)小技集」というテーマでLTを行いました。

<https://vscode.connpass.com/event/168760/>

LTの資料はこちらです。

LTでは以下の三つのテーマでSSHを使用したRemote Development拡張機能の活用法についてお話ししました。

- 踏み台(jumpbox)経由のリモート接続
- AWS Session Manger経由でのリモート接続
- リモート接続とLive Shareとの併用

資料中のリンクはこちらです。

##### 踏み台(jumpbox)経由のリモート接続

- [Can't connect through jumpbox · Issue #18 · microsoft/vscode-remote-release · GitHub](https://github.com/microsoft/vscode-remote-release/issues/18)
- [ProxyJump · Issue #1172 · PowerShell/Win32-OpenSSH · GitHub](https://github.com/PowerShell/Win32-OpenSSH/issues/1172)

##### AWS Session Manger経由でのリモート接続

- [AWS Systems Manager セッションマネージャーでSSH・SCPできるようになりました | Developers.IO](https://dev.classmethod.jp/articles/session-manager-launches-tunneling-support-for-ssh-and-scp/)
- [ステップ 7: (オプション) Session Managerを通して SSH 接続を有効にする - AWS Systems Manager](https://docs.aws.amazon.com/ja_jp/systems-manager/latest/userguide/session-manager-getting-started-enable-ssh-connections.html)

また今回はトップバッターでVSCodeの最新バージョンのアップデートについても話させていただきました。

世相が大変な状況の中オンライン開催の場を設けて下さったVS Code Meetupのみなさん、ありがとうございました！
