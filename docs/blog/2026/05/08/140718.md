# WSL上のffmpegでiPadで撮影した動画をクリッピング

## 前提

- 切り出す場所はVLCメディアプレイヤーで早回しして検討をつける

## コマンド例

### 通常の切り出し

- 撮影したときのファイル名にスペースがはいっているので引用符でかこむ

```
ffmpeg -ss 0:56:10 -to 0:56:48 -i "2026-05-01 08.50.27.mov" output_01.mp4
```

### インスタリール動画向け

iPadで横で撮っている場合に縦にクロップが必要な場合向け

```
ffmpeg -ss 0:56:10 -to 0:56:48 -i "2026-05-01 08.50.27.mov" \ 
-vf "crop=ih*9/18:ih" output_02.mp4
```

### 切り取る位置の調整


```
ffmpeg -ss 0:56:10 -to 0:56:48 -i "2026-01-01 01.50.27.mov" \
-vf "crop=ih*9/18:ih:(iw-ih*9/18)/2+iw/4:0" output_03.mp4
```

`iw/4` の部分の数字を変えて微調整。`iw/3` にすればもうちょい右、`iw/6` なら控えめにずれる感じ。

左にずれる場合は`+iw/4`が`-iw/4`になる。

```
ffmpeg -ss 0:56:10 -to 0:56:48 -i "2026-01-01 01.50.27.mov" \
-vf "crop=ih*9/18:ih:(iw-ih*9/18)/2-iw/4:0" output_04.mp4
```
