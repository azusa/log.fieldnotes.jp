# JJUG CCC 2019 Springで「大企業運営の法人向けサービスにおけるOpenJDK移行事例」という発表をしました。

2019/5/18のJJUG CCCで「大企業運営の法人向けサービスにおけるOpenJDK移行事例」という発表をしました。

以下、発表内での引用先へのリンクです。

- [[FELIX-6035] Allow urlhandlers to create urls for jrt protocol without an add-opens - ASF JIRA](https://issues.apache.org/jira/browse/FELIX-6035)
- [History for src/java.base/share/classes/java/net/URL.java - openjdk/jdk11u · GitHub](https://github.com/openjdk/jdk11u/commits/master/src/java.base/share/classes/java/net/URL.java)
- [[JDK-8213942] URLStreamHandler initialization race - Java Bug System](https://bugs.openjdk.java.net/browse/JDK-8213942)
- [OpenJDK 11.0.3 Released](https://mail.openjdk.java.net/pipermail/jdk-updates-dev/2019-April/000951.html)
- [Support | AdoptOpenJDK - Open source, prebuilt OpenJDK binaries](https://adoptopenjdk.net/release_notes.html)
- [Apache Tomcat 8 (8.5.41) - Security Considerations](https://tomcat.apache.org/tomcat-8.5-doc/security-howto.html)
- [1484079 – AWT java apps fail to start when stix-fonts is used as sans-serif font](https://bugzilla.redhat.com/show_bug.cgi?id=1484079)

- [[JDK-8188030] AWT java apps fail to start when some minimal fonts are present - Java Bug System](https://bugs.openjdk.java.net/browse/JDK-8188030)

登壇の中で話した、「OS等のアップデートで再起動を行う時の留意点」については昨日の [吉祥寺.pm18](https://kichijojipm.connpass.com/event/127917/)でお話ししているのでそちらをご覧下さい。(つまりこの発表は二つでワンセット)

<https://blog.fieldnotes.jp/entry/kichijojipm-18>
