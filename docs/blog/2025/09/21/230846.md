# Hyper-Vの内部ホスト内の仮想マシンにNATでアクセスできるようにする

## 接続をリストアップ

```
nmcli con show
```

## IPアドレス、サブネットマスク、ゲートウェイ、DNSを設定

```
sudo nmcli con mod eth0 ipv4.addresses 192.168.100.10/24
sudo nmcli con mod eth0 ipv4.gateway 192.168.100.1
nmcli con mod eth0 ipv4.dns 192.168.100.1
nmcli con mod eth0 ipv4.method manual
```

## 接続を再起動して設定を適用

```
nmcli con down eth0
nmcli con up eth0
```
