---
title: "複数のMarkdownファイルをPandocでPDF変換した後に1ファイルに連結する"
date: 2025-08-07
---

## スクリプト

```
export NOW=$(date +%s)
export LANG=ja_JP.UTF-8
pandoc $1 -o tmp/${NOW}.pdf --pdf-engine=lualatex -V documentclass=ltjsarticle -V geometry:a4paper -V listings=true -V listings-disable-line-numbers=false --listings -H <(echo '\lstset{breaklines=true,breakatwhitespace=true,basicstyle=\small\ttfamily,frame=single,numbers=left,numberstyle=\tiny,stepnumber=1,tabsize=2,showstringspaces=false,backgroundcolor=\color{gray!10}}')
```

## 実行

```
# *.mdのリストを出力
find docs/ > md2.txt
cat md2.txt  | xargs  -I XX  bash dopandoc.sh XX
cd tmp
pdfunite *.pdf hoge.pdf
```
