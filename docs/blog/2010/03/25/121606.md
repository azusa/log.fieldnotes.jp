# S2Dao-CodeGen 0.4.0 RC2をリリースしました。

### Bug

- [[DAOCODEGEN-64](https://www.seasar.org/issues/browse/DAOCODEGEN-64)] - org.seasar.codegen.lib.DateConditionElementのJavaDocの記述が誤っている問題を修正しました。
- [[DAOCODEGEN-65](https://www.seasar.org/issues/browse/DAOCODEGEN-65)] - S2Dao-Tigerのフェッチ機能が動作しない問題を修正しました。
- [[DAOCODEGEN-70](https://www.seasar.org/issues/browse/DAOCODEGEN-70)] - codegen-libのテスト実行時に、ログ出力が行われていない問題を修正しました。
- [[DAOCODEGEN-71](https://www.seasar.org/issues/browse/DAOCODEGEN-71)] - データベーススキーマからの生成時に、主キーの生成方法が未指定だとNullPointerExceptionが発生する問題を修正しました。
- [[DAOCODEGEN-72](https://www.seasar.org/issues/browse/DAOCODEGEN-72)] - h3データベースで指定するとNotTypeMatchExceptionが発生する型があった問題を修正しました。

### Improvement

- [[DAOCODEGEN-60](https://www.seasar.org/issues/browse/DAOCODEGEN-60)] - Mavenからの生成時、DBスキーマからの生成を可能にしました。
- [[DAOCODEGEN-61](https://www.seasar.org/issues/browse/DAOCODEGEN-61)] - JiemamyおよびDBスキーマからの生成時、主キーとシーケンスのマッピングを指定可能としました。
- [[DAOCODEGEN-67](https://www.seasar.org/issues/browse/DAOCODEGEN-67)] - サンプルコードとテストコードのディレクトリを分離しました。
- [[DAOCODEGEN-74](https://www.seasar.org/issues/browse/DAOCODEGEN-74)] - SQLファイル読み込みの際ののエンコードは、Javaソース出力時のエンコーディングを使用するようにしました。

### New Feature

- [[DAOCODEGEN-66](https://www.seasar.org/issues/browse/DAOCODEGEN-66)] - 戻り値が1件なメソッドで、SQLの実行結果が2件以上だった場合は、例外を投げるよう変更しました。

### Task

- [[DAOCODEGEN-75](https://www.seasar.org/issues/browse/DAOCODEGEN-75)] - Eclipse上における依存性管理の方法を、q4eからmaven-eclipse-pluginに変更しました。
