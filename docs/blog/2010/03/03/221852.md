# S2Dao 1.0.51をリリースしました。

- 変更点
  - Bug
    - [[DAO-151]](https://www.seasar.org/issues/browse/DAO-151) - MySQL用のページング機能（MySQLLimitOffsetPagingSqlRewriter）がバインド変数をもつSQLを考慮していない問題に対応しました。
    - [[DAO-154]](https://www.seasar.org/issues/browse/DAO-154) - org.seasar.dao.impl.ObjectResultSetHandlerにおいて、実際に使用しているクラスと違うクラスがテスト対象となっている問題を修正しました。
  - Improvement
    - [[DAO-153]](https://www.seasar.org/issues/browse/DAO-153) - 帰り値が1件なメソッドで、SQLの実行結果が2件以上の場合に警告ログを出している部分の処理をカスタマイズ可能にしました。

- 移行上の注意点

1.0.50からの移行上の注意点はありません。
