# S2Daoの逆襲〜漢ならDaoを書け！

世間ではBigtableをはじめとするKVSの話題が華やかで、RDBを使うにしてもDomaやS2JDBCやDBFluteなどのモダンなフレームワーク全盛ですが、堅くDaoを切る開発手法も捨てがたいと思うのです。

そして、S2Daoを使う上で問題となるのが、DaoやEntityの生成をどのように行うかということです。そのような開発者の悩みに応えるべく、S2Dao向けコード生成ツールS2Dao-CodeGenの新バージョンの開発を進めてまいりましたが、ドキュメントはまだまだですが、機能的には一通りそろったので、このブログの読者のみなさんにお披露目したいと思います。
