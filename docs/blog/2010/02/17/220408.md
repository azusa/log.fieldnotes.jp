# 入手方法

ZIPアーカイブは、こちら。  
<http://s2dao-codegen.sandbox.seasar.org/snapshot/>

mavenからのライブラリ使用は、以下の通り指定してください。

```
〜
  <repositories>
    <repository>
      <id>maven.seasar.org</id>
      <name>The Seasar Foundation Maven2 Repository</name>
      <url>http://maven.seasar.org/maven2</url>
    </repository>
    <repository>
      <id>maven.snapshot.seasar.org</id>
      <name>The Seasar Foundation Maven2 Repository(snapshot)</name>
      <url>http://maven.seasar.org/maven2-snapshot/</url>
    </repository>
  </repositories>

〜
    <dependency>
      <groupId>org.seasar.dao-codegen</groupId>
      <artifactId>codegen-lib</artifactId>
      <version>0.4.0-RC1-SNAPSHOT</version>
    </dependency>
```
