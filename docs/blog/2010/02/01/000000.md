# PocketWifiファーストインプレッション

今までモバイル環境にはauのW05Kを使用していたのですが、MacBookProを購入したことと、auの2年縛りが解けたのを機会に、イーモバイルのPocketWifiに乗り換えました。

まずこの製品の仕様として抑えておかなければならないこととして、連続通信時間4時間というバッテリー容量と、USB接続での通信時は充電不可という仕様があります。

自分の場合、会社の行き帰りでのiPhoneの通信環境と、PCにUSB接続での使用ができればよかったので、この仕様でも問題はありませんでしたが、一日の大半を外で過ごすような方には向かないと思います。バッテリーの持ちは使った印象としてはiPhoneにWifi接続して使用して3時間〜4時間といったところです。バッテリー表示が一目盛りになってから一時間以上持つので、思ったよりもバッテリーは持つ印象です。ただiPadの足回りとしてはバッテリーの持ちが弱点になるのではないでしょうか。

今のところ、iPhoneiは機内モードにして常にWiFi経由で接続しています。Phoneと組み合わせる場合ですが、iPhoneの電源を入れてからWiFiを認識するまで時間がかかります.  
また、WiFi接続がデフォルトだと10分無通信で切れるので、その後はPocketWifi側で接続操作をする必要があります。あと、WiFi接続が瞬断する場合があるので、Phoneの3G接続よりは機動性が落ちる感じです。あと、3G回線では全然だめだったYoutubeがいけるのでは？と思ったのですが、PocketWifiでも途中で再生がとまってしまうのは同じでした。PC経由では問題なく閲覧できるので、iPhone側の問題のようです...

  

結論としては、私のように、PC接続がメインで、そこにiPodTouchやiPhoneのためのWiFi接続がほしい人、ゲーム機やiPodTouchにWiFi環境を与えたい人には向いてますが、ヘビーユーザーのは電池の持ちがネックとなるのでリーチしづらいのではないかという印象です。
