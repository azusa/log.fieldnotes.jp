# S2Dao-CodeGen 0.4.1をリリースしました。

前バージョンからの修正は、以下の通りです。

- [[DAOCODEGEN-80](https://www.seasar.org/issues/browse/DAOCODEGEN-80)] - Jiemamyから入力したときに、カラムの論理名が設定されてないと、org.seasar.buri.common.util.template.exception.TemplateRuntimeExceptionが発生する問題を修正しました。
- [[DAOCODEGEN-81](https://www.seasar.org/issues/browse/DAOCODEGEN-81)] - バージョン番号およびタイムスタンプのプロパティ名指定が正しくない問題を修正しました。([Seasar-user:19952] )
- [[DAOCODEGEN-82](https://www.seasar.org/issues/browse/DAOCODEGEN-82)] - DBスキーマからの生成時、MySQLだとSQLRuntimeExceptionが発生する問題を修正しました。
