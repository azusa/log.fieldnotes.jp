# 第7回 xUtit Test Patterns 読書会

第7回を迎えたにもかかわらず3分の1くらいしかすすめない(汗)この読書会、第7回が11/14に行われました。

今回は[id:Youchan](http://blog.hatena.ne.jp/Youchan/)の発表を含め盛りだくさんだったのですが、個人的には文字列比較のテストのメリット・デメリットやDBを使う場合のテスト、Erattic Testの実例など自分の経験を踏まえて発言できたのがよかったと思います。

最後に[id:Youchan](http://blog.hatena.ne.jp/Youchan/)のデモがありました。Swingのロボット操作とTestNGが連動しているのは見物でした。

文字列化による比較がわかりづらかったみたいなので実例をあげときます。

<https://www.seasar.org/svn/sandbox/s2dao-codegen/trunk/codegen-core/src/test/java/org/seasar/codegen/impl/ExcelImportCodeDataSingleSheetTest.java>

の

```
        assertEquals(expect.toString(), actual.get("BuriDataPathHistory")
                .toString());
```

の部分が文字列化による比較を行っているところです。
